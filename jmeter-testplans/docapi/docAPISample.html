PrinterOn Enterprise API Tester



NOTE: Before you start, make sure you update the HTML in this file to point to your server.

To test the API, first enter an email address (or userID), the destination printer and select a document. Select Submit to start. Your job will be submitted to the server.

When the page reloads, the response is in XML, the browser may not parse the response so you may need to "View Source" to retrieve the Job Reference ID returned by the API.

To check the status of the job, enter the same user email (userID), destination printer name and the returned Job Reference ID into the Status Form. Submit to get the results.

The job is completed when a jobComplete indicates SUCCESS or FAILURE or a jobState of 7, 8 or 9 is returned.
